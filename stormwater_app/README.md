# Stormwater Report Generator — V1

A local Windows application for generating site-specific stormwater compliance
reports in Microsoft Word format. Built with Python and Streamlit.

---

## Quick Start (Windows)

**Option A — Double-click launcher (easiest)**
```
Double-click run.bat
```
This checks for Python, installs packages if needed, creates the template,
and opens the app in your browser automatically.

**Option B — Manual**
```bash
# 1. Install Python 3.10+ from python.org if not already installed

# 2. Install dependencies
pip install -r requirements.txt

# 3. Create the starter Word template
python create_template.py

# 4. Run the app
streamlit run app.py
```
The app opens at **http://localhost:8501**

## Workflow (Screen by Screen)

| Step | Screen | Purpose |
|------|--------|---------|
| 1 | Project Setup | Site name, client, dates, report type |
| 2 | Import Last Report | Upload previous PDF/DOCX for text reuse |
| 3 | System Selection | Add each BMP with ID, type, condition |
| 4 | Write-Ups | Edit findings, recommendations, maintenance |
| 5 | Photo Manager | Upload, order, and caption photos |
| 6 | Preview & Export | Review and generate the DOCX |

Use the sidebar to jump between steps at any time.
Click **Save** in the sidebar to persist your project to JSON.

## Customizing the Report Template

After running `create_template.py`:

1. Open `templates/report_template.docx` in Microsoft Word
2. **Add your logo**: Insert → Header → Edit Header → Insert Picture
3. **Edit header/footer**: Company name, address, phone, website
4. **Change colors**: Right-click a style in the Styles pane → Modify
5. **Change fonts**: Modify Heading 1, Heading 2, Normal styles
6. Save the file

The report generator loads your customized styles automatically at export time.
You never need to touch the Python code to rebrand the output.

To change colors in the Python generator directly (without editing Word),
edit `STYLE_CONFIG` in `app/services/report_builder.py`.

## Photo Caption Format

Captions are auto-built from structured fields:

```
{number}. {System Display Name} - {Component} - View {N}
```

Example:
```
1. Underdrain Soil Filter 1 - Outlet Structure - View 1
2. Underdrain Soil Filter 1 - Outlet Structure - View 1 (March 14, 2026)
3. Bioretention Cell 1 - Overall View - View 1
```

You can override any caption manually in the Photo Manager.

  Auto-fill from EXIF is V2.

## Common Issues

**"Module not found" on startup**
```bash
pip install -r requirements.txt
```

**Template not found on export**
```bash
python create_template.py
```
This regenerates the template. Your Word customizations will need to be redone.
Keep a backup of your customized template elsewhere.

**Photos not appearing in output**
- Make sure photos were added via the Photo Manager (not just referenced)
- Photos are copied to `projects/{id}/photos/` on upload — check that folder

**Port 8501 already in use**
```bash
streamlit run app.py --server.port 8502
```
